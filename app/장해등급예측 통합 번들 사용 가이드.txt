# 장해등급예측 통합 번들 사용 가이드

이 번들은 부상 부위/부상 종류/치료 기간/성별/나이/산업 분류/재해 유형 (총 7개의 int 값) + 장해 내용 (총 1개의 str 값)을 입력으로 받아 장해등급(1~15)을 예측하는 통합 모델입니다.

내부적으로는 다음 과정을 하나의 파이프라인으로 묶었습니다.
1. 정확문구 매칭(canonical)
2. BERT 기반 유사도 매칭
3. 위 둘을 충족하지 않을 때, 회귀 기반 번들 예측(Stage-1/2)

이 통합 번들을 하나의 함수 호출로 사용할 수 있습니다.

---

통합 번들 파일 : sanzero_integrated_bundle.joblib

파이썬 기준 사용 예시 : 
# 번들 불러오기
B = IntegratedBundle.load("/content/drive/MyDrive/sanzero/sanzero_integrated_bundle.joblib")
# 입력값 입력하기
payload = {
  "부상 부위":7,"부상 종류":4,"치료 기간":2,"성별":1,"나이":3,"산업 분류":2,"재해 유형":1,
  "장해 내용":"프레스 작업 중 손가락 절단"
}
# 예측 결과 출력하기
out = B.predict(payload)
print(out["predicted_grade"], out.get("message"))
